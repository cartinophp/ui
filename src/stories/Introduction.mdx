# @cartino/ui

Una libreria di componenti Vue 3 costruita con **Tailwind CSS**, **Reka UI** e **Tailwind Variants**.

## Struttura delle cartelle in Storybook

Per organizzare le storie in cartelle, usa la proprietà `title` con il separatore `/`:

- `"Components/Button"` → Cartella "Components" > Sezione "Button"  
- `"Forms/Input/Text"` → Cartella "Forms" > Sottocartella "Input" > Sezione "Text"
- `"Layout/Grid/Column"` → Cartella "Layout" > Sottocartella "Grid" > Sezione "Column"

### Esempio di struttura

```typescript
// Button.stories.ts
export default {
  title: 'Components/Button',
  component: Button
};

// Input.stories.ts  
export default {
  title: 'Forms/Input',
  component: Input
};
```
